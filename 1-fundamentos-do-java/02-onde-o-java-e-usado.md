# Capítulo 2: Onde o Java é usado (Aplicações reais e mercado)

## Por que Java?
Você escolheu uma das linguagens mais poderosas e requisitadas do mercado. O Java é a espinha dorsal de muitas empresas gigantes (bancos, seguradoras, aplicativos que você usa todo dia). Ele é robusto, seguro e foi feito para durar.

## Onde o Java vive hoje?
1. **Sistemas Bancários e Financeiros:** Se você usa o aplicativo ou o site do seu banco, é quase certo que existe código Java rodando nos servidores deles. O Java é extremamente seguro e confiável para processar transações financeiras.
2. **Aplicativos Android:** Até pouco tempo atrás, 100% dos apps Android eram feitos em Java. Hoje existe o Kotlin, mas o Java continua sendo a base de inúmeros aplicativos (e inclusive o Kotlin roda na mesma "máquina virtual" do Java!).
3. **Servidores (Back-end) de Grandes Empresas:** Netflix, Uber, Amazon, Mercado Livre e iFood utilizam muito Java em seus servidores, por trás dos panos, para aguentar milhares de acessos ao mesmo tempo sem travar.
4. **Minecraft:** O jogo Minecraft clássico (Java Edition) foi inteiramente escrito em Java!
5. **Sistemas Embarcados:** TVs inteligentes, roteadores, sistemas de painéis de carros, etc.

## O Mercado de Trabalho (Para quem faz ADS)
No curso de Análise e Desenvolvimento de Sistemas (ADS), aprender Java vai colocar você à frente de muita gente. O mercado tem uma "fome" gigante por programadores Java (nível júnior, pleno e sênior).
Grandes corporações têm sistemas legados (antigos) e novos sistemas sendo feitos em Java todos os dias. A remuneração para desenvolvedores Java costuma ser excelente!

## Conclusão
Aprender Java não é só aprender a programar, é aprender a criar **sistemas de verdade**. Requer um pouco mais de estudo inicial do que outras linguagens mais "soltas" (pois o Java tem regras mais rígidas para organizar o código), mas quando a base fica sólida, você aprende qualquer outra coisa muito rápido!
