# Capítulo 1: O que é Java?

## A Origem
O Java foi criado na década de 1990 por uma equipe liderada por James Gosling, na empresa Sun Microsystems (que hoje pertence à Oracle). O objetivo inicial era criar uma linguagem para programar eletrodomésticos inteligentes (como TVs e geladeiras), mas a linguagem acabou ganhando a internet e o mundo corporativo.

## O grande diferencial: "Write Once, Run Anywhere" (WORA)
O lema principal do Java é *"Escreva uma vez, rode em qualquer lugar"*. 
Isso significa que você pode escrever e compilar um programa Java em um computador com Windows e executar exatamente o mesmo programa em um Mac, Linux ou até em um celular Android, **sem precisar reescrever o código**.

**Como o Java faz essa mágica?**
Diferente de linguagens como C ou C++ (que traduzem o código diretamente para a linguagem específica da máquina física), o Java usa um intermediário:
1. Você escreve o código (arquivo `.java`).
2. O compilador do Java traduz esse código para algo chamado **Bytecode** (arquivo `.class`).
3. Esse Bytecode não é lido pelo seu sistema operacional, mas sim por uma "máquina virtual" chamada **JVM (Java Virtual Machine)**. 
4. Como existe uma JVM para Windows, uma para Mac e uma para Linux, o seu código roda em qualquer lugar que tenha a JVM instalada!

## Principais Características do Java
- **Orientada a Objetos:** Tudo no Java gira em torno de representar coisas do mundo real através de "Objetos".
- **Fortemente Tipada:** O Java é rigoroso. Você precisa dizer explicitamente qual é o tipo de dado que está usando (se é um número, um texto, etc.). Isso ajuda a evitar muitos erros (bugs).
- **Gerenciamento Automático de Memória:** O Java possui um recurso chamado *Garbage Collector* (Coletor de Lixo). Ele limpa automaticamente a memória do computador, apagando dados que seu programa não está usando mais.
- **Enorme Ecossistema:** Tem uma das maiores comunidades de desenvolvedores do mundo, documentação vasta e bibliotecas prontas para quase tudo.
