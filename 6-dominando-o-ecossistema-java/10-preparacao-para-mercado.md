## 1. O que estudar agora? (A Trindade do Sucesso)

### 🟢 Spring Boot
É o framework mais usado do planeta para Java. Ele usa tudo o que aprendemos (Maven, Lombok, Beans, Injeção de Dependência, Logs) para criar APIs de forma extremamente rápida.
**O que focar:** Spring Data JPA, Spring Security e Spring Web.

### 🟡 Banco de Dados e SQL
Você precisa saber salvar e buscar dados.
**O que focar:** PostgreSQL ou MySQL. Aprenda comandos `SELECT`, `JOIN`, `INDEX`, e como o Java usa o **Hibernate** para automatizar isso.

### 🔵 Docker e Nuvem (Cloud)
Hoje em dia não instalamos o sistema no Windows do cliente. Nós criamos um "Container" (Docker) e subimos para a nuvem (AWS ou Azure).

## 2. Como conseguir a primeira vaga?

1. **Crie um Portfólio no GitHub:** Não coloque apenas exercícios de aula. Crie um sistema real (ex: Um gestor de estoque, uma API de livraria, um sistema de tarefas).
2. **LinkedIn:** Coloque suas habilidades (Java, Spring, JUnit, Maven). Siga recrutadores e empresas de tecnologia.
3. **Certificações:** Se tiver oportunidade, a certificação oficial da Oracle (OCP) dá um peso enorme no currículo, mas não é obrigatória para começar.
4. **Soft Skills:** Saiba explicar como você resolveu um problema. Programação é 50% lógica e 50% comunicação com a equipe.

---

## 3. Desafio Final de Mestre
Para fechar esta documentação com chave de ouro, tente construir o seguinte projeto:

**"Sistema de Gestão de Biblioteca"**
- Use **Maven**.
- Crie uma classe `Livro` com **Lombok**.
- Use um `ArrayList` para guardar os livros.
- Crie métodos para Adicionar, Remover e Buscar por Nome.
- Use **Generics** para criar uma classe de repositório genérica.
- Salve os dados em um arquivo `.txt` ou `.json` usando **Jackson**.
- Crie **Testes Unitários** com **JUnit** para garantir que você não consegue adicionar um livro sem título.
