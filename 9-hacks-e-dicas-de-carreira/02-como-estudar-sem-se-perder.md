# Como Estudar Java Sem se Perder

O Java é gigante. Se você tentar aprender tudo de uma vez, vai desistir. Aqui estão os hacks para organizar seu cérebro.

## 1. Mapas Mentais
Em vez de ler páginas e páginas, desenhe caixinhas.
- `Variável` -> `Caixinha`
- `If` -> `Encruzilhada`
- `While` -> `Roda`
- `Classe` -> `Planta de Casa`

## 2. O Método Pomodoro
Estude por 25 minutos com foco total e descanse 5. O cérebro precisa desses 5 minutos para "gravar" o Java no seu HD interno.

## 3. Comente o seu código (Para o seu EU do futuro)
- **Hack:** Escreva comentários explicando o que você pensou.
```java
// Aqui eu usei um if porque só quero que menores de idade entrem se tiverem autorização
if (idade < 18 && temAutorizacao) { ... }
```
Daqui a uma semana, você vai olhar esse código e vai agradecer ao seu "eu do passado".

## 4. Pratique o Básico até ficar no Sangue
Não pule para Spring Boot se você ainda se confunde com um `for`.
- **Dica:** Faça o exercício da tabuada 10 vezes. Mude para ser tabuada de soma, de divisão. O cérebro precisa de repetição para automatizar o comando.

## 5. Participe de Comunidades
- Discord de programação, grupos de WhatsApp, LinkedIn.
- **Hack:** Tente ajudar alguém que sabe menos que você. Ensinar é a melhor forma de aprender.

O segredo não é a velocidade, é a **constância**. Um pouquinho por dia vale mais que 10 horas num domingo. ☕📚🚀
